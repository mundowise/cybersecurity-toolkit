# Ethical Use Policy

## Purpose

This software is developed for legitimate security research, education, and authorized authentication testing purposes. It is designed to help security professionals identify weak or default credentials in systems they are authorized to test.

## Acceptable Use

- Security assessments with **written authorization** from the system owner
- Educational purposes in controlled lab environments
- Personal learning and skill development on systems you own
- Authorized penetration testing engagements
- CTF (Capture The Flag) competitions
- Internal security audits by authorized personnel

## Unacceptable Use

- Accessing systems without explicit written permission
- Testing authentication systems of third parties without authorization
- Using this tool for credential stuffing against live production services
- Any activity that results in unauthorized access to systems or data
- Targeting systems you do not own or have explicit authorization to test
- Any activity that violates local, state, federal, or international law

## Legal Framework

Users must comply with all applicable laws, including but not limited to:

- **United States**: Computer Fraud and Abuse Act (CFAA), 18 U.S.C. § 1030
- **European Union**: Directive 2013/40/EU on attacks against information systems
- **United Kingdom**: Computer Misuse Act 1990
- **Canada**: Criminal Code Section 342.1
- **Australia**: Criminal Code Act 1995, Part 10.7

Unauthorized use of brute-force tools against systems you do not own constitutes a criminal offense in most jurisdictions and can result in significant civil and criminal penalties.

## Responsibility

The developers of this software assume **no liability** for misuse. Users are solely responsible for ensuring their use of this tool complies with all applicable laws and regulations. Always obtain written authorization before conducting any security test.

## Reporting Misuse

If you become aware of this tool being used for unauthorized or malicious purposes, please report it to the maintainers immediately via the contact information in SECURITY.md.
