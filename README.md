## Tools

| Tool | Description | Category |
|------|-------------|----------|
| [BackDoors_1.1](BackDoors_1.1/) | Modular remote access testing framework for authorized red team operations | Penetration Testing |
| [Brute-Force_1.1](Brute-Force_1.1/) | Multi-protocol authentication testing framework with GUI | Authentication Auditing |
| [WiFiNetScanner](WiFiNetScanner/) | Advanced modular WiFi and network reconnaissance suite | Network Analysis |
| [EncriptadoX](EncriptadoX/) | Cross-platform encryption and steganography desktop tool | Cryptography |
| [SecureMessenger](SecureMessenger/) | End-to-end encrypted multi-platform messaging system | Secure Communications |

## Requirements

- Python 3.10+
- Individual requirements per tool (see each project's README)

## Getting Started

Each tool is self-contained. Navigate to the tool directory and follow its README for installation and usage.

```bash
cd BackDoors_1.1/
pip install -r requirements.txt
python main.py --help
```
